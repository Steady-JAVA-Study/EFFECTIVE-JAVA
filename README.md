### Repository-Rule
 <br>
 
- **Branch naming convention** : <br>
   - 자신의 이름으로 branch를 만듭시다!
   - => git branch `자신의 이름` 
 <br>
 
- **Commit convention rule** : <br>
   - => `성명 : `챕터명` 학습 내용 정리` 
   - (ex) `김동윤 : Chapter 2 객체 생성과 파괴 학습 내용 정리` 
 <br>
 
- **Pull Request rule** : <br>
   - `Pull Request` 제목 : `김동윤 : `챕터명` 내용 정리 `
      - (ex) `김동윤 :  `챕터명` 내용 정리`<br><br>
